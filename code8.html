Garbage Collection -
A garbage collector is a tool that automatically manages the memory allocation and deallocation of your data structures
Underflow -
When a stack is empty (i.e. TOP= -1) and we try to delete more element from it, then this condition is called underflow condition
Overflow -
When stack is completely full (i.e. TOP= MaxSize -1 ) and we try to insert more element onto stack then this condition is called overflow condition
